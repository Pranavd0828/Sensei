# Components

React components for the Sensei MVP application.

## Purpose

Reusable UI components organized by domain and complexity.

## Structure

```
components/
├── ui/              # shadcn/ui base components (Button, Input, etc.)
├── auth/            # Authentication-related components
├── session/         # Session and step components
└── common/          # Shared application components
```

## Directories

### `ui/`
Base components from shadcn/ui:
- `button.tsx`, `input.tsx`, `card.tsx`, `dialog.tsx`, etc.
- These are generated by shadcn CLI
- Minimal customization needed

### `auth/`
Authentication components:
- `LoginForm.tsx` - Email input and magic link request
- `MagicLinkSent.tsx` - Confirmation message
- `AuthGuard.tsx` - Protected route wrapper

### `session/`
Session flow components:
- `SessionCard.tsx` - Display session summary
- `StepContainer.tsx` - Step wrapper with navigation
- `StepProgress.tsx` - Progress indicator (1-8)
- `ResultsScreen.tsx` - Scoring results display
- Individual step components

### `common/`
Shared components:
- `Header.tsx` - App header with navigation
- `LoadingSpinner.tsx` - Loading states
- `ErrorBoundary.tsx` - Error catching
- `StreakDisplay.tsx` - Streak pill
- `XPDisplay.tsx` - XP and level display

## Principles

1. **Composition Over Inheritance**: Build complex UIs from simple components
2. **Props Over State**: Prefer controlled components
3. **TypeScript**: All components must have typed props
4. **Accessibility**: Follow WCAG 2.1 guidelines
5. **Responsive**: Mobile-first design

## Example Pattern

```typescript
// session/StepProgress.tsx
interface StepProgressProps {
  currentStep: number
  totalSteps: number
}

export function StepProgress({ currentStep, totalSteps }: StepProgressProps) {
  return (
    <div className="flex items-center gap-2">
      {Array.from({ length: totalSteps }).map((_, i) => (
        <div
          key={i}
          className={cn(
            'h-2 flex-1 rounded',
            i < currentStep ? 'bg-primary' : 'bg-muted'
          )}
        />
      ))}
      <span className="text-sm text-muted-foreground">
        Step {currentStep} of {totalSteps}
      </span>
    </div>
  )
}
```
